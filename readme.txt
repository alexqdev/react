Project Name
ReactJS Card Matching Game


Product Purpose
A 2 player flip and match card game with 9 total cards. Built utilizing CSS GRID, SASS, ES6+ and ReactJS.


Product Scope
_Features
-ReactJS UI/UX components

_Rules
-objective: score more points than your opponent by matching as many cards as possible
-rules-
-a coin flip determines who goes first
-matched cards = +5 points
-joker = -5 points (do not lose turn)
-game is over once all cards are matched or game is ended manually
